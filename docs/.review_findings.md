## 第 1-10 轮裁决总表

### R2 L3日志/契约/配置
| 议题 | 裁决 |
|------|------|
| V1 日志 | Serilog 单 JSON Sink + CorrelationId(ThreadStatic) + 回调出口摘要 |

### R3 快照

### R4 Phase 1 范围
| 议题 | 裁决 |

### R5 IPC 安全
| 议题 | 裁决 |
|------|------|
| 认证 | PipeSecurity ACL + ImpersonateNamedPipeClient(写操作) + CreatorSid |
| HMAC | V1 不做 |
| 消息限制 | 64KB 硬上限 |
| 速率 | 全局 20 req/s burst 40 + 写 2 req/s burst 5 |
| 并发连接 | 8 |
| Stats | QueryAllStats 轮询 1Hz/5Hz，不推送，不用 MMF |
| Events pipe | V1 保留极简（4事件），或降级为单 pipe |

### R6 内存引擎
| 议题 | 裁决 |
|------|------|
| PagePool | V1 全局 ConcurrentStack（无 per-thread 缓存） |
| BatchLease | V1 堆 nint[]（无 ArrayPool） |
| 页清零 | V1 Return 时 NativeMemory.Clear（必须） |
| StatsCollector | V1 池级 Volatile.Read；IO 统计默认关或 padded Interlocked |
| 热路径计时 | V1 零计时；BDN/ETW 外置 |

### R7 文件系统
| 议题 | 裁决 |
|------|------|
| 三阶段写入 | 保留 Scan-Alloc-Commit，禁止双路径；Reserve 仅 SetLength |
| DirectoryTree | ConcurrentDictionary + 字典序 per-parent Lock（非全局 RWLS） |
| Rename | 双锁原子 pointer move + rollback |
| FspNotify | 解锁后同步，cache off 时 no-op；禁止 defer/异步队列 |
| SetFileSize | 全程写锁；Write Phase3 互斥；Phase3 页索引校验 |
| AllocationSize | 维护 _allocationSize 字段 >= FileSize；废弃 O(n) CountAllocatedPages |
